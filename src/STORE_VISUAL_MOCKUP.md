## KEY VISUAL ELEMENTS:

### Fixed Header (Always Visible)
- Clean white text on dark background
- Backdrop blur effect
- Back button left, title center, close right
- Subtle bottom border

### Progress Bar
- Animated fill from 0% to current %
- Purple/amber gradient
- Smooth easing curve
- Updates when themes purchased

### Shimmer Effect
- White gradient at 10% opacity
- Moves left to right across Complete Library card
- Repeats every 2.5 seconds
- 1 second pause between loops

### Pulsing Badge
- "Save $28.90" in green
- Scales from 1.0 to 1.1 and back
- Glow ring expands outward
- 2 second continuous loop

### Lock Shake
- Beneficiary locks wiggle -5° to +5°
- Happens every 3 seconds
- 4 second delay between shakes
- Draws attention to upgrade

### Ceremony Stagger
- Each ceremony card fades in sequentially
- 0.05 second delay between each
- Slides in from left (-10px)
- Smooth easing
